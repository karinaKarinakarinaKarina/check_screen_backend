Фронтэнд для данного проекта: https://github.com/LizaEmelyanova/screen-check 
Для корректной работы приложения необходимо запустить сервер Next.js и сервер Django. 
Запуск сервера: python manage.py runserver \ http://localhost:8000 - главная страница с основным функционалом 
http://localhost:8000/api/yolo-model-info/ - вывод классов модели для проверки корректной загрузки модели
